dt-skill-creator

Create, modify, and evaluate Dynatrace-safe skills with built-in security guardrails. Use this skill whenever someone wants to create a new skill for the Dynatrace/SPINE ecosystem, improve an existing skill, run evals to test skill quality, benchmark skill performance, or optimize a skill's description for better triggering. This skill enforces safety rules — no external URLs, no dangerous shell commands, no data exfiltration patterns — so every skill it produces is safe for internal Dynatrace environments. Also use when reviewing skills for security compliance, or when the user mentions "dt skill", "safe skill", "Dynatrace skill", or wants to package a skill for distribution within the team.

Install

# Copy directly into your project
git clone https://github.com/samuel-sanchez-moreno/aisteroids.git .aisteroids
cp -r .aisteroids/skills/dt-skill-creator .github/skills/
# Or use a git submodule for automatic updates
git submodule add https://github.com/samuel-sanchez-moreno/aisteroids.git .aisteroids
ln -s ../.aisteroids/skills/dt-skill-creator .github/skills/dt-skill-creator
