dt-presentation-builder

Use when building Dynatrace-branded HTML presentations, slide decks, reports, dashboards, or executive briefings from existing markdown, notes, or data. Triggers: "build a presentation", "create slides", "HTML slide deck", "executive briefing", "status update deck", "convert this report to slides", "Dynatrace presentation", "Strato slides", "executive summary HTML".
